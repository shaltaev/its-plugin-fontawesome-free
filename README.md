# its-plugin-fontawesome-free

Plugin for icons-to-sprite, extractor for fontawesome-free

## Getting Started

### Installing

Add to your nodeJS project

```sh
npm install -D its-plugin-fontawesome-free
```

or

```sh
yarn add -D its-plugin-fontawesome-free
```
